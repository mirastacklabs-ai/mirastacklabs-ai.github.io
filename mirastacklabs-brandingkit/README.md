# MIRASTACK LABS Branding and Marketing Kit

This kit is derived from the current live website identity and codifies a full B2B enterprise brand system for MIRASTACK LABS.

## What is included

- Visual identity and logo usage pack
- Color, typography, and design tokens
- Positioning, messaging, and narrative architecture
- B2B marketing playbook for platform engineering buyers
- Ready-to-use templates for web, social, email, and sales collateral

## Immediate usage checklist

1. Use the horizontal dark logo on dark surfaces and hero areas.
2. Use the horizontal light logo on white or light documents.
3. Use the stacked dark logo for social avatar and square placements.
4. Keep all UI implementations mapped to `brand-tokens.css`.
5. Align all copy with the messaging framework before campaign launch.
